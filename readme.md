## Instalar virtualenv

pip install virtualenv

### Crear un entorno virtual

virtualenv venv

activar entorno virtual en windows

.\venv\Scripts\activate.ps1

o en cmd

.\venv\Scripts\activate.bat


## Instalar las librerias

pip install -r requirements.txt
